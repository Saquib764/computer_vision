1. Installing OpenCV

Posted on Sept 6, 2017 20m OpenCV Python Ubuntu

Installation on Ubuntu or Debian

OpenCV can be installed either from the Ubuntu or Debian repository or from the official site. This does not install other important openCV modules like Aruco, SIFT, SURF etc. To build OpenCV with opencv_contrib modules, please follow this tutorial.

OPTION 1: Install from repository

Use below command to intall OpenCV from repository:

sudo apt-get install libopencv-dev python-opencv

Its important to note that Python3 bindings does not exist in this repository.

OPTION 2: Install from official site

Installation from official site is a bit lengthy but easy and installs latest version. But I found a bash script by milq that does all the hardwork for us. Follow the following steps.

Download the installation script install-opencv.sh.

Content of script:

Once downloaded the script, open your terminal and excute the following command:

bash install-opencv.sh

and simply wait for it to complete installation.

Check for installed version

To check for verion of opencv installed, execute following command to start python shell:

python

Then run the following python script

>>> from cv2 import __version__
>>> __version__
'3.3.0'

and you are ready to rock. Enjoy! :)
