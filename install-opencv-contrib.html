1. Installing OpenCV with contrib module

Posted on Sept 17, 2017 40m OpenCV Python OpenCV_contrib Ubuntu

Installation on Ubuntu or Debian

OpenCV has many non-free modules which are included in this contrib.

Update your Ubuntu/Debian

sudo apt-get -y update
sudo apt-get -y upgrade
sudo apt-get -y dist-upgrade
sudo apt-get -y autoremove

Install dependencies

Simply copy paste all the lines at once.

sudo apt-get install -y build-essential cmake
sudo apt-get install -y qt5-default libvtk6-dev
sudo apt-get install -y zlib1g-dev libjpeg-dev libwebp-dev libpng-dev libtiff5-dev libjasper-dev libopenexr-dev libgdal-dev
sudo apt-get install -y libdc1394-22-dev libavcodec-dev libavformat-dev libswscale-dev libtheora-dev libvorbis-dev libxvidcore-dev libx264-dev yasm libopencore-amrnb-dev libopencore-amrwb-dev libv4l-dev libxine2-dev
sudo apt-get install -y libtbb-dev libeigen3-dev
sudo apt-get install -y python-dev python-tk python-numpy python3-dev python3-tk python3-numpy
sudo apt-get install -y ant default-jdk

Download/clone OpenCV

Download .zip stable verion of OpenCV from here. Download the Source Archive and unpack it into your working directory.

OR

You can also clone the latest OpenCV from its Git repository.

cd ~/my-working-directory
git clone https://github.com/opencv/opencv.git

Clone OpenCV contib modules

Clone the cutting-edge opencv_contrib module into your working directory.

git clone https://github.com/opencv/opencv_contrib.git

Build OpenCV from source using CMake

Create a temporary folder inside your opencv source folder. For example build

cd ~/my-working-directory
cd ./opencv-YOUR-VERSION
mkdir build
cd build

Configure your opencv build to include opencv_contrib modules.

cmake -DOPENCV_EXTRA_MODULES_PATH=../../opencv_contrib-master>/modules ..
make -j4

-j4 run this process in 4 cores in parallel. Thus, if you have more core for example 8 then imply use thi code to get it done faster. However, this will take sometime in general.

make -j8

Install the build

Once above code is completed, install the binaries.

sudo make install

Check installation

Test you installed version by importing modules from opencv_contrib in python shell.

>>> import cv2.aruco as aruco

This should compile without any error.

And as usual, Enjoy! :)
